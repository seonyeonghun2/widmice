   3) 카카오 지도 삽입

      - 카카오 개발자 사이트 ==> 개인회원 가입,   결제카드 등록은 필수 아님!

      - 카카오 MAP (https://apis.map.kakao.com/)

      - 1일 300,000회 사용 제한


    3-1) 카카오 개발자사이트 (https://developers.kakao.com) 접속 [가입 후 로그인]
    3-2) 개발자 등록 및 앱 생성 : 내 애플리케이션 클릭!
    3-3) 웹 플랫폼 추가: 앱 선택 – [플랫폼] – [Web 플랫폼 등록] – 사이트 도메인 등록
    3-4) 사이트 도메인 등록: [웹] 플랫폼을 선택하고, [사이트 도메인] 을 등록합니다. (예: http://localhost:8080)
    3-5) 페이지 상단의 [JavaScript 키]를 지도 API의 appkey로 사용합니다.
    3-6) 웹이나 앱을 실행합니다.
    3-7) KAKAO Maps API 페이지 > Guide > 준비하기가 모두 끝났다면, 이제 시작하기 대로 진행합니다.
    ※ 처음부터 지도 코드가 삽입된 HTML 샘플이 제공되지만, 직접 만들어서 사용해보세요!
    ※ 더 많은 지도활용 예제는 KAKAO Maps API 페이지 > Sample 항목을 선택해 확인하세요!
        (https://apis.map.kakao.com/web/sample)
    ※ 구글보다 네이버, 다음(카카오) 지도의 정확도가 (국내에선) 높습니다.
